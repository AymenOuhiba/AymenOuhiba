> *Opta-level football analytics from any match video — free, local, no cloud needed.*

**Highlights:** 1M rows processed · Multi-backend (Ollama / OpenAI / Claude) · Read-only SQL validation · CSV & report export

`Python` `Flask` `Ollama` `Llama 3.2` `SQL Server` `PostgreSQL` `SQLite` `LangChain`

### 🧠 [Grad-ECLIP — Explainable CLIP via Gradient-based Attention](https://github.com/AymenOuhiba/Grad-ECLIP-Explainable-CLIP-via-Gradient-based-Attention-Analysis)

`CLIP` `PyTorch` `Vision Transformers` `Grad-CAM` `ImageNet` `Jupyter` `XAI`
